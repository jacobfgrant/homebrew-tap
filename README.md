# homebrew-tap

Homebrew formulae for tools by [@jacobfgrant](https://github.com/jacobfgrant).

## Installation

```bash
brew tap jacobfgrant/tap
```

## Available Formulae

### unity-rearm

Convert x86_64-only Unity macOS games to native ARM64 on Apple Silicon.

```bash
brew install unity-rearm
```

See the [unity-rearm repo](https://github.com/jacobfgrant/unity-rearm) for documentation.
